# Ed's Quest Log

A personal productivity and point tracker built to gamify daily tasks.

## Features
- Earn points for completing tasks (hygiene, chores, errands, medical appointments, etc.)
- Track progress toward rewards (free hours, free days, purchase list items)
- Full task history log
- Works offline, saves progress automatically

## Rewards
- 20 pts → 1 Free Hour
- 50 pts → 3 Free Hours  
- 100 pts → Free Day or Purchase List Item

## Built with
Plain HTML, CSS, and JavaScript. No dependencies.
